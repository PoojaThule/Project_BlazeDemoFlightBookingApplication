1. Project: Automation Testing of Flight Booking Application using Selenium WebDriver
2. Objective: The objective of this project is to automate the flight booking functionality of the BlazeDemo web application using Selenium WebDriver, Java, and TestNG. The project aims to validate positive and negative test scenarios, implement the Page Object Model design pattern, and generate detailed reports with logging and screenshots to ensure application quality and reliability.
3. Technology used: 
• Programming Language: Java 
• Web Automation: Selenium WebDriver
• Test Framework & Assertions: TestNG 
• Build & Dependency Management: Maven 
• Design Pattern: Page Object Model (POM)
• Logging: Log4j2

4. Reporting:-
• TestNG default reports 
• Extent Reports
